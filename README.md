# VÉLO & VINTAGE — Demo E-Commerce

Premium urban streetwear e-commerce application built for portfolio showcase.

## Stack
- **React 18** + **React Router v6**
- **Vite** (dev server on `localhost:3000`)
- **Recharts** for admin analytics
- **Lucide React** for icons
- **BroadcastChannel API** for cross-tab state sync
- **LocalStorage** for persistence

## Getting Started

```bash
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000)

## Pages
| Route | Description |
|-------|-------------|
| `/home` | Storefront homepage |
| `/men` | Men's category — filtered catalog |
| `/women` | Women's category — filtered catalog |
| `/cart` | Shopping cart |
| `/checkout` | 3-step checkout |
| `/login` | Login / demo access |
| `/admin/dashboard` | Admin panel |

> **DEMO NOTICE**: This is a portfolio demonstration only. No real orders, payments, or data are processed.
